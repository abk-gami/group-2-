HTML Tag Use
h1 to h6 used used to define HTML headings
p defines a paragraph
i The content inside is typically displayed in italic
em used to emphasize a text
strong used to indicate an important text
cite used for the title of a text
span is an inline container used to mark up a part of a text, or a part of a document
strike to define strikethrough text
